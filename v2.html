Spielanleitung

Spielprinzip:

Fusioniere gleichfarbige Blasen mit derselben Zahl, um Blasen höherer Stufen zu erzeugen.

Steuerung:

  • Tippe auf eine Blase, um sie auszuwählen.
  • Tippe auf ein leeres Feld, um sie dorthin zu bewegen.
  • Tippe auf eine gleichwertige Blase, um beide zu fusionieren.
  • Nach jedem Zug erscheint automatisch eine neue Blase.
  • Drücke auf "Neue Blasen", um zusätzliche Blasen ins Spiel zu bringen.
  • Alle 3 Züge erscheint eine Bonusblase!
  • Drücke auf "Blasen neu mischen", wenn du keine Züge mehr hast (kostet Punkte!).

Spezialblasen:

  • Patrick (P): Spezialblase, kann mit Rott fusioniert werden.
  • Rott (R): Spezialblase, kann mit Patrick fusioniert werden.
  • Wenn Patrick und Rott fusioniert werden, entsteht eine Level-3-Blase!

Tipps:

  • Plane voraus! Schaue auf die nächsten Blasen.
  • Versuche, Combos zu erzeugen, indem du Kettenfusionen auslöst.
  • Halte genug freie Felder für neue Blasen.
